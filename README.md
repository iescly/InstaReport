# 📦 Installation

<details>
<summary><b>📱 Android / Termux</b></summary>

```bash
pkg update && pkg upgrade
pkg install python git

git clone https://github.com/iescly/InstaReport
cd InstaReport

pip install -r requirements.txt
python InstaReport.py
```

</details>

<details>
<summary><b>🖥️ Windows</b></summary>

```bash
1. Go to the GitHub Releases section
2. Download the latest Windows executable (.exe)
3. Extract the archive if needed
4. Run the application as Administrator
```

> ⚠️ Some antivirus software or Windows Defender may display false-positive warnings due to executable packing and automation-related behavior. If this happens, verify the file source and allow the application manually.

</details>

<details>
<summary><b>🐧 Linux / Kali</b></summary>

```bash
sudo apt update
sudo apt install python3 python3-pip git

git clone https://github.com/iescly/InstaReport
cd InstaReport

pip3 install -r requirements.txt
python3 InstaReport.py
```

</details>
